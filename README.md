# ⚖️ منظومة العميد للتسويق القانوني الآلي (2026)

هذا المستودع يمثل "المحرك التسويقي" لمنصة العميد. يقوم النظام بجمع إيميلات العملاء المحتملين في المملكة العربية السعودية آلياً، ثم إرسال حملات بريدية ترويجية لبوت العميد الذكي.

## 🚀 كيف تعمل المنظومة؟

يعمل النظام بشكل آلي بالكامل عبر **GitHub Actions** وفق التسلسل التالي:
1. **Scraper (الصياد):** يقوم بمسح محركات البحث يومياً لجلب إيميلات شركات ومحامين وعقاريين.
2. **Database (المخزن):** يتم تحديث ملف `emails.txt` بالأهداف الجديدة دون تكرار.
3. **Striker (المسؤول):** يقوم بإرسال رسائل العرض المجاني (7 أيام) عبر بريد Outlook الرسمي.
4. **Auto-Commit:** يتم حفظ النتائج وتحديث القائمة تلقائياً كل صباح.

## 📂 هيكلة الملفات

| الملف | الوظيفة |
| :--- | :--- |
| `scraper.py` | محرك البحث عن الإيميلات (Leads Generation). |
| `striker.py` | محرك الإرسال (Email Automation). |
| `emails.txt` | القائمة السوداء للأهداف (Database). |
| `daily_hunt.yml` | عقل النظام الذي يدير التشغيل اليومي. |

## 🛠️ إعدادات التشغيل (Setup)

عشان تشغل النظام، لازم تتأكد من الآتي:

1. **السرية التامة:** أضف الـ 16 حرف (App Password) في إعدادات المستودع:
   - اذهب إلى `Settings` > `Secrets and variables` > `Actions`.
   - أضف Secret جديد باسم `MS_APP_PASS`.
2. **رابط البوت:** تأكد من وضع رابط بوت تليجرام الخاص بك داخل ملف `striker.py`.

## ⏰ جدول التشغيل
النظام مبرمج للعمل تلقائياً يومياً الساعة **8:00 صباحاً بتوقيت مكة المكرمة**.
